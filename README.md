рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћ Flask-рЃўрЃА **send_file()** рЃЏрЃћрЃЌрЃЮрЃЊрЃў.

```python
    app.run(host='0.0.0.0', port=8000)
```

## **5. Dockerfile рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃў**

```docker
# рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћ рЃърЃљрЃбрЃљрЃарЃљ рЃќрЃЮрЃЏрЃўрЃА Python image

# рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃцрЃљрЃўрЃџрЃўрЃА рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ
```

рЃљрЃЦ **рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃўрЃџрЃў рЃћрЃарЃЌрЃўрЃљрЃюрЃљрЃЊ** рЃЊрЃљрЃЋрЃљрЃЎрЃЮрЃърЃўрЃарЃћ `COPY . /app/`, рЃарЃљрЃф **Docker Image-рЃўрЃА рЃљрЃњрЃћрЃЉрЃљрЃА рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃА рЃ«рЃЊрЃўрЃА**.
